# Password Generator

## Description

Password Generator is a web application that creates random secure passwords.

Users can customize password generation using:

- Uppercase letters
- Lowercase letters
- Numbers
- Special characters
- Password length


## Technologies Used

- HTML5
- CSS3
- JavaScript


## Features

- Random password generation
- Input validation
- Copy password option
- Responsive design
- User-friendly interface


## How To Run

1. Download or clone repository

2. Open folder

3. Run index.html in browser
